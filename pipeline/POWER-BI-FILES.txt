🆘 TROUBLESHOOTING
────────────────────────────────────────────────────────────────────────

"File not found" error
  → Check file path in M code (must be absolute: C:/Users/...)

Relationships won't create
  → Verify column data types match (text to text, date to date)

Measures show wrong values
  → Use Deals table for calculations, not dimension tables

Hierarchies don't work
  → Set sort orders (Stage by StageOrder, Market by MarketOrder)

Visuals are blank
  → Check for IsOpen = TRUE filter on pipeline visuals
